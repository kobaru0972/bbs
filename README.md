# 機能一覧
* 見出しの折りたたみと展開機能
* 投稿機能
* 検索機能
* 管理者ページへのログイン機能
* 管理者ページからの削除機能

# 使用方法
* 記事の検索
    1. 記事の検索エリアをクリックする
    2. 展開された入力フォームに検索した文字列を入力する
    3. 「検索」ボタンをクリックする
    ![serch](https://user-images.githubusercontent.com/92970448/140683027-ba39027f-e0ac-4cc9-b8c4-2177a9450ce5.PNG)

* 記事の書き込み
    1. 新規記事の書き込みエリアをクリックする
    2. 展開された入力フォームのお名前、件名、メッセージ欄すべてに入力する
    3. 「書き込む」ボタンをクリックする
* 記事の削除
    1. 画面右下の「管理者ページ」のリンクをクリックする
    2. ユーザー名とパスワードを入力し、「ログイン」ボタンをクリックする
    3. 削除する記事のIDとパスワードを入力し、「削除」ボタンをクリックする
